单行函数note

一.字符串: 1.字符截取和连接: 1).substr:取字符串str中的指定字符，从位置a开始取长度为b的字符串，假如a为正则从左边开始，否则从右边开始 select dname, substr(dname,-3,2),substr(dname,3,2) from DEPT where dname='ACCOUNTING'; 2).concat(str1,str2)---将字符串str1与str2连接起来(也可以通过'||'号直接相连) 3).instr(str,'z',pos,nth)---取得str字符串中从左边开始每一次出现z字符的下标位置(下标从1开始),参数为字符串,'待搜索字符',开始搜索位置,第几个搜索到得字符; INSTR(str,find_string[,start][,occurrence]) 在str中查找字符串find_string，然后返回find_string所在的位置,可提供一个可选的start位置来指定该函数从这个位置开始查找。 同样，也可以指定一个可选的occurrence参数，来说明应该返回find_string第几次出现的位置 select dname, instr(dname, 'N',4,2) from DEPT where dname='ACCOUNTING'; 4).仅对两边的字符起效,中间的字符无效 LTRIM(x,[,trim_string] 从左边截去一些字符,可选参数trim_string指定要截去的字符;如果不指定,则默认截去空格。 RTRIM(x,[,trim_string] 从右边截去一些字符,可选参数trim_string指定要截去的字符;如果不指定,则默认截去空格。 TRIM(x,[,trim_string] 两边同时截去一些字符,可选参数trim_string指定要截去的字符;如果不指定,则默认截去空格。 select ltrim(' soar is a boy ') lt, trim(' soar is a boy ')t, rtrim(' soar is a boy ')rt from dual;

5).REPLACE(x,search_string,replace_string) 在x中查找search_string，并将其其替换为replace_string。 select replace(' soar is a hero ', ' ' , '#') from dual;

2.大小写转换: upper(str)---将字符串str全部转换成大写 lower(str)---将字符串str全部转换成小写 initcap(str)---将字符串中每个单词的首字母大写 select initcap('soar is a hero') from dual;

3.填充: lpad(str,12,'')---左填充，即将字符串str长度填充到12，假如其不足12位则在左边以号填充 rpad(str,12,'*')---右填充，同上

4.字符串长度: length(str)---计算字符串str的长度

5.ascII码和字符的转换 ASCII(X) 返回字符x的ascII码 CHR(x) 返回ASCII码的x的字符 select ascII('a'), chr(97) from dual;

二.数字函数 1.精度截取: round(45.926,2)---将前一数保留指定的小数位，并四舍五入(45.93)，假如指定位是负数则意为在小数点左边保留指定位, round(45.923,-1)=50 小数点左边保留1位,即可想象成4.5923,四舍五入后至5,再乘以10,即得到50 round(45.93,-2)=0 小数点左边保留2位,即可想象成0.4593,四舍五入后至0,再乘以100,即得到0 round(55.93,-2)=100 小数点左边保留2位,即可想象成0.5593,四舍五入后至1,再乘以100,即得到100 round(45.923,0)=46

2.截取小数位: trunc(45.926,2)---同上，只是不四舍五入(45.92) 3.取模,求余: mod(1600,300)=100 4.取绝对值:ABS SQL> select abs(100),abs(-100) from dual;

  100       100

3.日期函数: 1).取得月份最后一日,last_day(date) select last_day(to_date('2010-04', 'yyyy-mm')) from dual; select last_day(to_date('2010-04-01', 'yyyy-mm-dd')) from dual;

2).加上n个月后的日期add_months(date, n) select add_months(to_date('2010-01-03', 'yyyy-mm-dd'), 6) from dual;

3).下个周几的日期next_day(date, '星期五') select next_day('10-5月-2001','星期五') next_day from dual; select next_day(sysdate,'星期五') next_day from dual;

4).计算日期间月份差months_between(date1, date2) select round(months_between('18-12月-1999','19-3月-1999'),0) mon_between from dual;

5).计算不同时区时间new_time(date, str_时区, str_格式) select to_char(sysdate,'yyyy.mm.dd hh24:mi:ss') bj_time, to_char(new_time(sysdate,'PDT','GMT'),'yyyy.mm.dd hh24:mi:ss') los_angles from dual;

6).日期的round和trunc:对日期进行四舍五入:对day进行round,round至最近的周日日期; round,存在上进位 Select sysdate, ROUND(Sysdate,'YEAR') ry, ROUND(Sysdate,'month') rm, ROUND(Sysdate-3,'day') rd From DUAL; trunc将不上进,均截取掉多余部分 Select sysdate, TRUNC(Sysdate,'YEAR') ry, TRUNC(Sysdate,'month') rm, TRUNC(Sysdate,'day') rd From DUAL;

四.通用函数 1.nvl(expr1,expr2)---expr1为空则显示expr2,否则显示expr1 select nvl(comm, empno),comm, empno,mgr from emp where empno in (7369,7499,7839);

2.nvl2(expr1,expr2,expr3)---expr1不为空则显示expr2,否则显示expr3 select nvl2(comm, empno,mgr),comm, empno,mgr from emp where empno in (7369,7499,7839); Select SAL,COMM,NVL2(COMM,SAL+COMM,SAL), sal+comm From EMP; (null+任何值为null)

3.nullif(expr1,expr2)---二个相等则显示空符,否则显示expr1 select nullif(1,3-3), nullif(1,3-2) from dual;

4.coalesce(expr1,expr2,...,exprn)---从expr1开始依次找到不为空的expr，找到就显示，直到最后，否则显示exprn select coalesce(null, null,8,0) from dual;

两个集合比较： INTERSECT/MINUS :

Intersect和Minus的操作和Union基本一致，这里一起总结一下： Union，对两个结果集进行并集操作，不包括重复行，同时进行默认规则的排序； Union All，对两个结果集进行并集操作，包括重复行，不进行排序； Intersect，对两个结果集进行交集操作，不包括重复行，同时进行默认规则的排序； Minus ,对两个结果集进行差操作，不包括重复行，同时进行默认规则的排序。 可以在最后一个结果集中指定Order by子句改变排序方式。
