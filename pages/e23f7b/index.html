VuePress构建博客-vdoing

# 安装使用 (强制格式规范化,自动形成侧边栏,强烈推荐)

  • 安装vuepress
npm install -D vuepress
1
  • 下载主题 vdoing github (opens new window)
  • 解压后,安装依赖模块module
npm install
1
  • 按约定格式将文档拷贝到docs目录下
  • 运行
npm run dev
1

# 推荐理由:

  • 注重配置大于约定,文档按约定格式组织后,即可自动形成侧边栏;(另外，强制规范可约束规范归类,提高文档的可读性)
  • 支持git直接部署;
  • 支持中文名称md文件的访问;
  • 已集成一键copy代码,搜索插件;

# 图片静态资源

  • markdown安装 Snipaste软件进行截图; (Ctrl+F12截图)
  • 建议适用vscode编辑markdown文件,安装paste image插件来从剪切板按一定规则黏贴图片到md文件,可设置Shift+F12快捷键；
  • 设置paste image插入链接的方式: ${imageSyntaxPrefix}./${imageFilePath}${imageSyntaxSuffix}

注意: 建议静态资源放在md文件的路径进行直接管理; 方便预览，也方便后续部署到服务器后仍能正常展示,也方便区分具体图片使用情况; 此方式仅在编译打包后才会生效,本地run dev状态下,image无法展示;

# 自动部署

# 本地nginx部署

#!/usr/bin/env sh

# 确保脚本抛出遇到的错误
set -e


# git提交地址，也可以手动设置，比如：push_addr=git@github.com:xugaoyi/vuepress-theme-vdoing.git
#push_addr=`git remote get-url --push origin`
push_path="/usr/local/Cellar/nginx/1.19.8/html"
#push_path="/usr/local/var/www"
# 打包生成的文件夹路径
dist_path="docs/.vuepress/dist"
echo "开始本地部署: $dist_path -> $push_path "

# 生成静态文件
npm run build

deploy_path=$push_path/dist
if [ -d $deploy_path ];then
    rm -fr $deploy_path
    echo "clear nginx files: $deploy_path ..."
fi

if [ -d $push_path ];then
    cp -r $dist_path $push_path
    if [ $? -ne 0 ]; then
        echo "update fail: $?, $dist_path -> $push_path ";exit
    fi
    echo "update ok: $dist_path -> $push_path "
fi

if [ -d $dist_path ];then
    rm -rf $dist_path
    echo "clear $dist_path done"
fi

nginx -s reload
echo "reload nginx ..."

# 远程giteepage部署


#!/usr/bin/env sh

# 确保脚本抛出遇到的错误
set -e


# git提交地址，也可以手动设置，比如：push_addr=git@github.com:xugaoyi/vuepress-theme-vdoing.git
#push_addr=`git remote get-url --push origin`
push_addr="git@gitee.com:soar0216/notepages.git"
commit_info=`git describe --all --always --long`
# 打包生成的文件夹路径
dist_path=docs/.vuepress/dist
# 推送的分支
push_branch=master

# 生成静态文件
npm run build

# 进入生成的文件夹
cd $dist_path

git init
git add -A
git commit -m "deploy, $commit_info"
git push -f $push_addr HEAD:$push_branch

cd -
if [ -d $dist_path ];then
    rm -rf $dist_path
    echo "clear $dist_path done"
fi
