# 搜索引擎使用技巧

# 搜索方式：

  • 单词整体搜索,"关键字",注意需要加引号;
  • 加入"-"进行过滤指定网站;
    • 如：红烧肉 -下厨房 -美食天下 -家常菜谱网
  • 指定在哪个网站搜索，实现站内搜索
    • 如：钢铁侠 site:v.qq.com
  • 指定文件类型搜索：
    • 如： CSS filetype:ppt
  • 指定标题/内容
    • 如：离骚 intitle/intext 赏析
  • 查汇率:
    • 100rmb =?hkd
  • 数学：
    • graph for sin(x)
  • 直接输入"天气"，查看天气
  • 输入“法国时间”，直接查看当前时间和日期，时区

    • 建议使用英文去github搜索
