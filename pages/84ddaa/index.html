# isync

由于isync名称冲突,其可执行程序调整为mbsync

  • ~/.mbsyncrc
IMAPStore chenyang@erayt.com-remote
Host imap.exmail.qq.com
Port 993
User chenyang@erayt.com
PassCmd "pass chenyang@erayt.com"
AuthMechs LOGIN
SSLType IMAPS
CertificateFile /etc/ssl/certs/ca-certificates.crt
PipelineDepth 10  #有时候不加这一行, 邮箱并发太大, 会被服务器拒绝掉

MaildirStore chenyang@erayt.com-local
Subfolders Verbatim
Path /home/cy/.local/share/mail/chenyang@erayt.com/
Inbox /home/cy/.local/share/mail/chenyang@erayt.com/inbox
Trash /home/cy/.local/share/mail/chenyang@erayt.com/Trash

Channel chenyang@erayt.com
Far :chenyang@erayt.com-remote:
Near :chenyang@erayt.com-local:
Patterns *erayterayt
SyncState *
Create Both
Expunge Both
CopyArrivalDate yes  #将原邮箱的时间信息也同步过来, 否则以拉取的时刻上传, 不方便
Sync All
