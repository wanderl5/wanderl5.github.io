vi操作

# VI操作

# 快速打开头文件

  • gf: jump to file
  • copen: 打开make控制展示界面; make后的结果可展示于此

# VI键盘操作图

# vi命令规则：

数字 + 命令(改，删，复) + 文本 1-n c，d，y 移动命令

命令 移动到

gg 文件头 G 文件尾

0 行头 $ 行尾

5| 光标所在的第5列 5G 第5行

/pattern 向下--模式 y/pattern/+ 复制从光标到模式所在行的下一行 ?pattern 向上

n 与命令同方向 N(shift+n) 与命令反方向

块操作: 按v进入可视化模式,对其进行操作,上下；操作仍可加上执行次数; 标记+移动: 标记移动起始点ma, 移动光标到移动结尾点,按v进入可视模式,按'a选定起始点至当前行,执行操作即可

# 可视化的几种操作

键 作用
v 常规可视模式，从当前列开始，支持按行，列选中
shift + v 行可视模式，从当前行开始，支持按行选中
Ctrl + v 类似v，但支持列选中

# 各类操作

# 标记

- mx	标记
- 'x'	返回标记行的首字符,单引号
- 'x'	返回到标记行的确切地址，行和列tab
- ''	切换标记地

### 打开文件的参数：
	vi a b c 3个文件同时打开，再使用:n/N,:e#文件间切换,:x退出多文件阅读
	+2 打开文件时，光标在第2行；

### 模式：
	一般模式：刚打开时的模式；此模式可剪切、复制，粘帖，删除，翻页，搜索，替换
	编辑模式：i/I，a/A，o/O进行插入，c/C,s/S,r/R进行更新
	指令模式：w，q存盘、退出，以及打开另外的文件

### 光标方向控制（移动）：

~~~bash
前/后 /上/下：h/l /j/k;
行首/尾：0/$;
文件首/尾：gg/G;
翻页：Ctr+F/Ctr+B往后/前；
:2  跳到第2行；
:$  跳到文件的最后一行；等同于G；（：0等同于gg）

CTRL+f/b	前后滚动一屏
CTRL+d/u	半屏上下滚动

  k          上
h   l      左  右
  j          下

^        移动到该行第一个非空格的字符处
w        向前移动一个单词，将符号或标点当作单词处理
W        向前移动一个单词，不把符号或标点当作单词处理
b        向后移动一个单词，把符号或标点当作单词处理
B        向后移动一个单词，不把符号或标点当作单词处理
0        到行首
$        到行尾
gg       到页首
G        到页末
行号+G   跳转到指定行
Ctrl+g   查询当前行信息和当前文件信息

fx       向右跳到本行字符x处（x可以是任何字符）
Fx       向左跳到本行字符x处（x可以是任何字符）

tx       和fx相同，区别是跳到字符x前
Tx       和Fx相同，区别是跳到字符x后

C-b      向上滚动一屏
C-f      向下滚动一屏
C-u      向上滚动半屏
C-d      向下滚动半屏
C-y      向上滚动一行
C-e      向下滚动一行
~~~


### 复制，删除，剪切，粘贴：(组合数字+命令+移动命令)

~~~bash
p：粘帖；（小写p），粘帖到光标所在行的下面；

Y：复制光标所在的那一行；
nyy：复制光标所在行算起的n行，并放入缓存区；

x：删除光标所在的字符；

D：删除从光标所在位置到行尾的所有内容，相当于d$；
dw：删除光标所在算起到该单词末尾的所有内容；
dd：删除整行
dG: 删除从光标到末尾
ndd：删除从光标所在行算起n行的内容；

"xyw: 块操作，缓存光标所在单词到缓存x，可用"xp，取出使用；x可为a-z任意字符，注意开头的“号，缓存引用符；所有c,d,y及其命令组合均可使用此法。
1,5 ya a：在ex模式下，使用1,5 ya a，即把1到5行yank到寄存器a中，可使用p或者“ap取出使用；	
:5,6 m 12 -》 5-6行移动到12行后面(co, 拷贝)
~~~

## 重复操作

```
.：重复上一次操作， 在ecs前所有操作
@:  宏
```

### 搜索：

~~~bash
/或?:启动搜索
/^13:30:只在每行行首搜索13:00；
/13:30$:在行尾找13：30，可用于找行尾空格："/ $",没报pattern not found表示有部分行尾有空格，有空格可能导致某些语句执行错误。
/\^:转义，查找特殊字符，如^,$,*
~~~

### 搜索替换：
~~~bash
:1,20 s/soar/raos 从1-20行搜索soar并替换成raos；20改成$表示从首行到末尾；
在一行内替换头一个字符串 old 为新的字符串 new，请输入 :s/old/new
在一行内替换所有的字符串 old 为新的字符串 new，请输入 :s/old/new/g
在文件内替换所有的字符串 old 为新的字符串 new，请输入 :%s/old/new/g
进行全文替换时询问用户确认每个替换需添加 c 选项，请输入 :%s/old/new/gc 
~~~

### 插入：
~~~bash
a/A：光标后/光标所在行的最后开始插入;
i：光标所在处插入；
o：光标所在行的下方新建一行开始插入；

r ./chap10/10.1.c 读文件到光标位置处
r !date		读系统date命令到光标位置
~~~

### 撤销操作：
~~~bash
u： 撤销最后执行的命令
U： 修正之前对该行的操作
u2：回复第二次修改，存于#2号缓存(1-9为系统预留的删除缓存，a-z可由用户自定义，通过“访问)
显示状态：
Ctr+g：显示文件名，所在行，总行数，百分比；相当于:f;
Ctrl+R：Redo
~~~

### 保存和退出：
~~~bash
:w 存盘；
:w!强迫存盘；
:wq 存盘并退出；相当于ZZ；
:w file1 另存为file1；
:1,3w file 第1到3行存为新文件file
:6,$w >>file 第6到文件尾所有行追加到file

:q 退出；（若有改变将不能用此命令退出）；
:q!强迫退出并不保存文件，在q不能退出时，用此命令；
~~~

### 多文件切换：
~~~bash
:e# 在备用文件和当前文件之间切换，只能两者之间切换
:e ./chap10/10.1.c 在当前编辑session中编辑另一文件，此法可利用缓存”实现文件间的文本移动
:n/N 向前/后在多个文件间移动
:x 可直接保存并退出多文件编辑，多文件移动过程中，均要求先保存文件，或者用e!放弃所有修改，才能移动
set相关命令(vi设置)
set nu : number行号
set ic : ignore capital忽略大小写
set ai : automatic indent 自动缩进

:set hls is（hlsearch(高亮)，incsearch(随搜索内容变化焦点位置)）
取消set，用set no+相关命令
~~~

### 大小写切换

- `~`大小写切换
- 移动切换

| 指令 |       含义       |
|------|------------------|
| `gu` | 小写             |
| `gU` | 大写             |
| `g~` | 大小写切换toggle |

- `gUw`: 单词全大写
- `gU$`:  整行大写 `g~$` 整行切换大小写
- `gUG`: 全文大写

> 任意移动命令均可配合使用
> replace with g, 当前行使用s来替换,尝试成功后使用`g&`,全量应用上次的s替换动作;
> 在vi可使用`help g`查看帮助

### `J`合并行

- `J` 单行合并,会带空格
- `v` + up/down select + `gJ`: 合并不带空格


### 打开文件

- `gf`: 打开当前行,当前单词的文件

### 特殊用法：

- 重复输入同一字符：一般模式下，输入“重复次数+i/a+重复字符+ESC”，如重复输入50星号可用”50i*“再按ESC；输入50个”我“字就替换掉*；

### 内置命令：

~~~bash
"Delete every line from the current line (.) to the end-of-file ($)...
:.,$delete

"Replace "foo" with "bar" everywhere in lines 1 to 10
:1,10s/foo/bar/

"Center every line from five above the current line to five below it...
:-5,+5center
类似的,有left,right命令

"nmap 表示 “normal-mode key mapping”。nmap 之后的 <silent> 选项使得映射不能回传它所执行的任何命令

对应的有imap,insert模式下的Map
~~~

### 设置高亮

~~~ bash
:set hlsearch    设置高亮
:set nohlsearch  关闭高亮
:nohlsearch      关闭当前已经设置的高亮
~~~

### fg快速回到修改点

- `CTRL+z`当前vi编辑进入后台
- `fg`回到最近的修改
